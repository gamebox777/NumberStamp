# NumStamp (React版)

画像に番号付きのスタンプ（丸印、角丸四角）や矢印、矩形を配置し、作業記録を残すためのWebツールです。
Python版のNumStampをReact + KonvaでWebアプリケーションとして移植しました。

## 機能

- **画像読み込み**: ドラッグ＆ドロップまたはファイル選択で画像を背景として読み込みます。
- **スタンプ配置**: クリックで連番スタンプを配置。色、サイズ、形状（円、角丸四角）を変更可能。
- **矢印描画**: スタンプを選択すると表示されるハンドルをドラッグして、スタンプから矢印を引けます。
- **矩形描画**: ドラッグして矩形（枠線）を描画。
- **画像保存**: 編集した内容を画像としてダウンロードできます。

## 開発環境のセットアップ

```bash
npm install
npm run dev
```

ブラウザで `http://localhost:5173` にアクセスして確認してください。

## デプロイ (Vercel)

Vercelにデプロイする場合は、リポジトリをVercelにインポートするだけで自動的にビルド設定が認識されます。

- Build Command: `npm run build`
- Output Directory: `dist`
- Install Command: `npm install`

## 操作方法

1.  **画像を読み込む**: 画面右上の画像アイコンをクリックするか、ファイルをウィンドウに直接ドラッグ・アンド・ドロップします。
2.  **ツール切り替え**: 左側のツールバーでツールを選択します。
    - **選択 (Select)**: スタンプや矩形の移動、削除（Deleteキー）、属性変更。
    - **スタンプ (Stamp)**: クリックしてスタンプを配置。
    - **矩形 (Rectangle)**: ドラッグして矩形を描画。
3.  **設定変更**: 右側のパネルで、選択中のオブジェクトまたは新規追加するオブジェクトの色やサイズ、番号を設定します。
4.  **保存**: 左側のダウンロードアイコンをクリックして画像を保存します。
