Number Stamp（ナンバースタンプ）をリリースしました

作業手順書やバグ報告、デザインレビュー画像にサクッと番号や注釈を入れたい。そんな思いから開発した画像注釈Webアプリをリリースしました。 インストール不要・ブラウザだけで完結するので、どなたでもすぐに使い始められます。 🌐 アプリURL： https://number-stamp.vercel.app/ 🐙 GitHub： https://github.com/gamebox777/NumberStamp 📝 ブログ： https://unitygamebox.com/  

どんなアプリなの？

Number Stamp（ナンバースタンプ）は、画像に連番スタンプ・矩形・テキスト・ペン・矢印などの注釈を自由に配置し、作業手順書やバグ報告書、レビュー画像などを素早く作成するためのWebアプリケーションです。 専用ソフトのインストールは一切不要。ブラウザを開けばすぐ使えます。  

どんな機能が使えるの？

現在、以下の6種類のツール（モード）が利用できます。

🔢 スタンプ（Stamp）

クリックした場所に連番スタンプを自動で配置します。 手順書のステップ番号付けやバグ番号の振り分けが一瞬で完了！ ・番号・増分（Step）を自由に設定 ・円・角丸四角から形状を選択 ・形状色・文字色を個別に設定可能

🔲 矩形（Rectangle）

ドラッグで矩形（四角い枠）を描画します。 注目させたいエリアを囲んで強調したいときに最適です。 ・枠線の色・太さ・塗りつぶし・破線スタイルを設定可能 ・矩形内にテキストを入力可能

🔤 テキスト（Text）

画像上の任意の位置にテキストコメントを追加します。 スタンプの説明文や補足事項を書き込むのに便利です。

✒️ ペン（Pen）

フリーハンドで自由に線を描画できます。 矢印や囲みが大雑把でも、ペンならではの素早さが魅力です。

📏 直線・矢印（Line / Arrow）

始点から終点へドラッグして直線・矢印を引けます。 スタンプ間を矢印で繋いで順序を示すことも可能です。

👆 選択（Select）

配置したオブジェクトの選択・移動・削除ができます。 Ctrlキーを押しながらクリックすると複数選択も可能です。  

ショートカットキー

よく使うキーボード操作もサポートしています。 ・Ctrl+Z：元に戻す（Undo） ・Ctrl+Y：やり直し（Redo） ・Ctrl+C：コピー ・Ctrl+V：貼り付け ・Delete / Backspace：削除  

使い方はとてもシンプル

1. アプリにアクセスする 2. 画像をドラッグ＆ドロップ（またはツールバーのアイコンからファイル選択）して背景にセット 3. 左のツールバーでモードを選んでクリック・ドラッグで注釈を追加 4. ツールバーのダウンロードアイコンから PNG・JPG 形式で書き出し 以上！インストールもサインアップも不要です。  

保存・書き出しについて

・画像を保存（Export）：PNG または JPG 形式で高解像度書き出し ・プロジェクト保存（Save）：作業状態を JSON ファイルとして保存。後でそのまま再編集が可能 ・プロジェクト読込（Load）：保存した JSON を読み込んで作業を再開  

プライバシーについて

このアプリはすべてブラウザ内で完結しており、バックエンドへのデータ保存（データベース）は一切行っていません。 アップロードした画像や注釈データがサーバーに保存されることはないので、安心してご活用いただけます。
